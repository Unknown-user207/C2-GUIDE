# Shadow Guide — Your Silent AI Mentor

A complete, modular AI companion with memory, project awareness, offline fallback, and a web interface. Built for developers who value privacy, persistence, and presence without noise.

## Features
- **Silent by default** — no voice, just text
- **Persistent memory** — recalls past conversations
- **Project awareness** — sees your files and structure
- **Offline fallback** — works without internet (Ollama)
- **Multi-model support** — OpenAI, Grok, DeepSeek, Local
- **VPS-ready** — deploy to any cloud with Gunicorn
- **Web interface** — optional browser chat

## Quick Start
```bash
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python shadow_guide_complete.py
```
