# Knowledge Base — Shadow Guide

This folder stores all manual knowledge, research, and insights for your projects. The Guide automatically loads every `.md` and `.txt` file here.

## How to Use

1. **Add a file**: Drop any `.md` or `.txt` file here.
2. **Read a file**: Type `read filename.md` in the Guide.
3. **Search**: Type `search keyword` to find anything across all files.
4. **Add a note**: Type `addnote Your note here` — it will be saved to `manual_notes.md`.
5. **List all files**: Type `knowledge` to see what's available.

## Recommended Files

- `c2_architecture.md` — Design decisions for your C2.
- `android_insights.md` — Research on Android clients.
- `deployment_notes.md` — VPS, Cloudflare, Docker.
- `lessons_learned.md` — What broke and how you fixed it.

## Philosophy

This folder is your *external brain*. The more you feed it, the smarter your Guide becomes.
