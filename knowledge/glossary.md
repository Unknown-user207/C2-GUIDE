# 📖 Glossary & Terminology

## Core Terms

### C2 (Command & Control)
The infrastructure used to control compromised devices.

### Agent
The Android application deployed on target devices.

### Beacon
Periodic communication from agent to C2 server.

### Payload
The malicious/agent software delivered to target.

### Stealth
Techniques used to avoid detection.

### Persistence
Methods to maintain access after reboot.

### OPSEC
Operational security measures.

## Android Terms

### APK
Android Package Kit - application package format.

### AAB
Android App Bundle - modern Android package format.

### ADB
Android Debug Bridge - development tool.

### Root
Superuser access on Android.

### Magisk
Popular root solution.

### Bootloader
Initial program that loads the operating system.

### DEX
Dalvik Executable - Android bytecode.

### Obfuscation
Code transformation to hinder reverse engineering.

## Communication Terms

### HTTPS
HTTP over SSL/TLS - secure communication protocol.

### WebSocket
Full-duplex communication protocol.

### FCM
Firebase Cloud Messaging - Google push service.

### DNS Tunneling
Using DNS queries for data exfiltration.

### Certificate Pinning
Hardcoding trusted certificates.

## Security Terms

### EDR
Endpoint Detection and Response.

### AV
Antivirus software.

### Sandbox
Isolated environment for testing.

### Forensics
Investigation of digital evidence.

### Honeypot
Decoy system to detect attackers.

## Development Terms

### Kotlin
Modern Android development language.

### Coroutines
Kotlin's concurrency framework.

### Flow
Kotlin's reactive streams.

### Gradle
Android build system.

### ProGuard
Code obfuscation tool.

## Operational Terms

### IOC
Indicators of Compromise.

### TTP
Tactics, Techniques, and Procedures.

### ATT&CK
MITRE's knowledge base of adversary tactics.

### IoA
Indicators of Attack.

### Kill Chain
Stages of an attack.

## Attack Terms

### Reconnaissance
Information gathering phase.

### Exploitation
Taking advantage of vulnerabilities.

### Persistence
Maintaining access.

### Privilege Escalation
Gaining higher privileges.

### Lateral Movement
Moving through the network.

### Exfiltration
Data theft.

### Cover Tracks
Hiding evidence of compromise.

## Stealth Terms

### Anti-Forensics
Preventing evidence collection.

### Anti-Debugging
Preventing dynamic analysis.

### Anti-VM
Avoiding virtual machine detection.

### Process Hollowing
Replacing legitimate process.

### DLL Injection
Loading malicious code.

### Rootkit
Hidden malicious software.

## Legal Terms

### Authorized Testing
Testing with explicit permission.

### Red Team
Authorized penetration testing team.

### Bug Bounty
Rewarded vulnerability disclosure.

### Responsible Disclosure
Ethical vulnerability reporting.

### Security Research
Academic/industry security investigation.
